## Animation Features

### Visual Elements
- **Point Numbering**: Sequential numbers show weld order
- **Color Coding**: Different colors for each weld type
- **Path Lines**: Connected lines show weld sequence
- **Legend**: Color-coded legend explains weld types
- **Scaling**: Automatic coordinate transformation and centering

### Weld Type Colors
- **Normal**: Blue (`#000080`) - Standard structural welds
- **Frangible**: Red (`#FF0000`) - Breakaway welds for temporary seals
- **Stop**: Purple (`#800080`) - Pause points for user intervention
- **Pipette**: Magenta (`#FF00FF`) - Filling operation points

## Generation Commands

```bash
# Generate PNG animation
python -m microweldr -weld design.svg -png_animation preview.png

# Generate SVG animation
python -m microweldr -weld design.svg -png_animation preview.svg

# Generate both G-code and animation
python -m microweldr -weld design.svg -g_out output.gcode -png_animation preview.png
```

## Technical Details

- **Image Size**: 800x600 pixels with 50px margins
- **Format**: PNG with RGB color space
- **Font**: Arial with fallback to system default
- **Coordinate System**: Automatic scaling and centering
- **Point Radius**: 3 pixels with black outline
- **Line Width**: 2 pixels for path connections

## File Formats Supported

- **PNG**: Raster format for high-quality static visualization
- **SVG**: Vector format for scalable animated sequences (work in progress)

These examples demonstrate MicroWeldr's capability to visualize complex weld sequences, making it easier to verify weld patterns before sending to the printer.
